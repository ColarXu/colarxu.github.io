坊间传言，[Github](https://github.com/)  是全球最大的同性交友网站👬，我偶尔也会进去瞧两眼。但毕竟不是搞技术的，就像女生进 Gay bar 一样，玩不出什么花样。

这周在学习用 [Hugo](https://gohugo.io/) 搭建自己的博客，最后一步，把本地文件夹部署到 Github，需要在 Terminal（终端）上输入几个 ``git`` 命令。其实，照着网上的教程，简单几步就能搞定。但我这人有个毛病，当不了纯粹的**执行机器**。你说要这么做，但不说清楚为什么要这么做，我就一肚子问号，非得弄明白了才心安。

所以，今天临时决定把博客放一放，好好学一下 Git——反正后面肯定会用到。

我选择了 [廖雪峰的 Git 教程](https://www.liaoxuefeng.com/wiki/896043488029600)，结合**费曼学习法**，每学完一部分，就用自己的语言「传授」一遍，以巩固自己的学习成果。这一系列文章，既是我的学习笔记，也是我站在廖雪峰的肩膀上🙇🏻，以文字工作者的视角，重新编写的 Git 教程。

就这样吧，**重新开始系统学习，重新开始坚持写作**，是以为记。
