饭团的初夏

某天，卢奶奶下班回家，手里拎了个袋子，我问是啥，她神神秘秘地说，是她做饭团的小玩意。 我当她是一时兴起，想做日式饭团玩玩，做个两三次，估计热情就会褪去。没想到，从此以后，我每天的午餐都变成了饭团。再也不用操心做什么剩菜料理了。 离游手好闲，好吃懒做的人生又迈进了一步，感恩。 据我暗戳戳的观察，卢奶奶每晚睡觉前，就会在电饭煲里备好做饭团的主材料：大米、绿豆/红豆。第二天，当我还像猪一样昏睡在床上时，她便已经起来淘米、煮饭，之后再洗脸、刷牙、拉鸡屎。 等吃完早餐，饭也煮好了。这时候，卢奶奶会去打开锅盖，用饭勺把饭搅松，散一散热，再把饭舀进去她的小玩意————饭团模具里，中间加点肉松，再铺一层饭，压成型。最后，在成型的饭团表面撒一层香松，包上海苔。午饭就做好了。 有时候，卢奶奶会带一个饭团去单位当午餐，但大多数时候，像是专门为我准备的。 中午只吃饭团，没有配菜，看起来挺寡的。实际吃起起，倒不觉得。可能是因为加了肉松，可能是因为海苔本身有咸味，也可能饭是里面加了豆子，嚼起来有一股香味，又或者是这三者的合力所致。 清清爽爽、简简单单、满满足足，卢奶奶的饭团，就这样解决了我一顿又一顿的中午饭。 只有一次遗憾。那天，卢奶奶给我做了两个饭团，一大一小，吃完大的，肚子饱饱的，小的便留到了晚餐。可惜，卢奶奶下班回来，拿起小饭团一闻，说酸了。本以为饭团可以就放，看来，还是要抓紧吃。

等雨下来

醒来胸闷气短，走到窗户前，深吸一口外面的空气，鼻子里肺里都没感到新鲜。一查彩云天气，好吧，今明两天下雨。 雨将下未下，在天上默默地压着，人胸口里的气也不快活。总不能坐在这里，跟它沆瀣一气。于是，站起身来，铺好被子，灶上蒸两个叉烧包，煮上茶，洗抹布，拖地。 屋子收拾完，便坐下吃包子喝茶，完了剥个橘子，可惜酸。接下来刷牙洗脸，牙齿细细刷了，脸从额头往下一寸一寸擦洗，眼镜也洗得干干净净。 人是清爽了，走到阳台，打了一套八段锦。五月第一天，算是好的开始。 卢奶奶说启迪去了金泽玩，想看看雨中的古镇，可惜现在雨还没下。是呢，闷了一早上了，现在还没下。 现在是下午三点半，天黑得像傍晚一样。据说，还有二十分钟，雨就要落下来了。 把厚衣服收拾了，一件件叠好，塞进真空压缩袋，抽成扁扁，衣柜一下子空出不少。但离理想状态还差很远，衣服还是太多。 于是，暗暗下了个决心：今年一整年，什么衣服裤子鞋子都不准买！

肩膀疼痛的早晨

醒来已经是 9 点 10 分了。奇怪，今天睡得沉，睡得死，连卢奶奶在床边晨运的声音都没听到。 右边肩膀一直疼，手一捏，肌肉紧得像块石头，不知道是不是昨晚写毛笔字弄的，如果是，那也一定是我的姿势有问题。 微信给卢奶奶推了个公众号，有趣的是，被推的那个公众号，在简介里写着「我讨厌算法推荐」之类。号的主人是个曾经的产品经理，现今的无业游民，倒是跟我有点像。粉丝不多，阅读数在 200 左右，内容跟阮一峰的科技爱好者周刊差不多，把自己看到比较有意思的东西，整合在一起。突然觉得这个做法不错，一方面把自己看到的，喜欢的东西整理了出来，不至于随风消逝；另一方面，也具有分享价值。创作总会遇到瓶颈，不当作者，当个编辑也不错。 不过，订阅号不能跳转外链，想看推荐的文章，点一下是不可能的，得拉到文末的参考文献，复制文章链接，再去浏览器打开阅读，体验太差。微信果然封闭，这东西还是适合放到 Web 上玩。不知道作者有没有开放订阅，等下问问卢奶奶，我去 RSS 他一下。 今天的计划是继续看读库，现在已经消灭了三本，如果专心一点呢，今天估计是可以全部看完的，比原计划超出了很多时间。转念一想，又觉得不大好，怎么能把读书当成计划呢？读书，还是得不求多，不求快。此处须自我警戒。 哎，肩膀太痛，贴个膏药去。 顺便夸一下卢奶奶，临走前给我削了一个苹果，装在碗里，摆在桌上；装葡萄干的小盒子也补满了。真是把我当猪养。 现在还不到下午三点，六本读库已全部看完。说是看完，其实好几篇文章只是快速翻阅，没进眼也没进脑。以后不能把读书当成任务，再次警戒。

一些书债

上个月底，卢奶奶给我下了一道「读书令」，让我在规定时间内，把她准备卖掉的书看完。我满口答应下来，说一个月时间就行了，DDL 一到，没看完的书，她也可以卖掉。 好吧，一个月已经过了快三分之一，我还没开动。之前都去干嘛了呀？ 以下是我的回忆流水帐： 我先是沉迷于 Github 上 Ada 学院的编程课程，发现太多东西不懂，为了打基础，去 Codecademy 上把 Command Line 的课程过了一遍。接下来的课程是学习使用 Git，可不知道为什么，我突然对乔布斯产生兴趣，便暂停课程，去把《乔布斯传》啃完了。 在忍受完乔布斯过山车一样的脾气后，我的兴趣又转移到了皮克斯，毕竟他们做的才是实打实的创意。读完《孵化皮克斯》，发现里面讲的都是商业运作（作者是 CFO），而不是创意。顺藤摸瓜，找到皮克斯的创始人之一艾德·卡特姆写的《创新公司：皮克斯的启示》，才算找对书，看了头两章，确实对创意人员的管理有很多借鉴之处。然而，这本书让我莫名进入了一种「工作状态」，我可是无业游民哦，还是等工作以后再看吧，暂缓之。 之后，我在网上冲浪时，发现一个名叫无产阶级图书馆的电子资源网站，兴致勃勃下了小几十本书，忍不住开始看（完全忘记卢奶奶给我的任务了）。先是看了一本政治八卦书，写得像校园小说，有些地方明显在胡说八道，浪费我的好奇心；随后是《暗网》，看了几章，没多大意思，弃；最后开读张纯如的《美国华人史》，这是本好书！ 读完《美国华人史》，忽然想起卢奶奶布置的任务，赶紧把本月要看的书搬了出来，整理如下： 《读库 1901》 《读库 1902》 《读库 1903》 《读库 1904》 《读库 1905》 《读库 1906》 《一课经济学》 《新黑暗时代》 读库 19 年系列，其实是去年的书债了，也不知怎的，书里的一些文章，当时就是看不下去，所以今年也没有续订。虽然有 6 本，但部分文章去年已经看过了，只要补补漏，把之前落下的篇章看完，估计实际的阅读量在 3 本书左右。 《一课经济学》，想看了好几年，这段时间却没啥欲望看它。不行，作为奥地利经济学派的入门书，还是得看。 最后一本《新黑暗时代》，是某个周末跟卢奶奶出门散步，特地去图书馆借的，总得在到期前看完吧，不枉那次走一遭。 好了，梳理完毕，去看书咯。

由爆浆玉米想到的

卢奶奶喜欢吃玉米，尤其是爆浆玉米。今天早上，她蒸了两根当早餐，我们一边啃一边开始呱呱呱： 我：这玉米好吃是好吃，就是颗粒有点小，一口咬下去，牙齿还没爽够就触底了，粒再大一点就好了。 卢：没有啊，我觉得挺好吃的，它的皮很薄，不像其他玉米粒的皮那么厚，我喜欢吃这种。 我：我们谈论的不是同一个问题。 卢：那你说。 我：这个玉米粒的皮是比较薄，水分也多，口感还很清甜，这些都是优点。但我想说的是，它的玉米粒还不够大，也就是说肉比较少，吃的时候意犹未尽。如果它的颗粒再大一点会更好。 卢：可能它就是这样子的。 我：对，虽然它现在是这个样子，但未来可以继续改良品种。你想想，如果爆浆玉米的肉又多，汁水又充足，会受到市场欢迎的吧。 卢：可我觉得现在的没问题。 我：额……如果是在工作环境中，这样回答很讨厌。如果有个同事跟你说一个新想法，你回答「我觉得没问题」，那么讨论还没开始就终止了。换位思考一下，如果你跟同事说一个新想法，同事这样回你，你也会觉得讨厌的吧。因为对方把一个「自己的主观感受」当作「反对原因」塞给你了。换句话说，就是论据不充分。 好吧，我们现在在餐桌前，又不是工作环境。卢奶奶肯定很讨厌我 😂 但「爆浆玉米」这个产品确实不错，简单总结一下它的优势： 识别度高：爆浆玉米是乳白色的，一眼就能跟黄玉米、黑玉米区别开来。蒸熟后的爆浆玉米，还有一种和田玉的光泽（这句文案是我加的）； 口感独特：玉米粒皮薄、汁水充足、口感清甜，跟有嚼劲的糯玉米不一样，走的是小鲜肉路线； 名字起得好：吃起来真的会爆浆，很形象； 价格厚道：两根爆浆玉米 9.9 元，比普通玉米贵不了多少。 美中不足的就是，颗粒不够大。可能大一点以后，口感会变老？我不知道。 最后，引用一句卢奶奶的话： 所谓爆浆玉米，就是玉米粒里的水会喷得满脸都是。
