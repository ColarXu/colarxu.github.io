一些书债

2020-04-08 · 43 words · 许仙人

上个月底，卢奶奶给我下了一道「读书令」，让我在规定时间内，把她准备卖掉的书看完。我满口答应下来，说一个月时间就行了，DDL 一到，没看完的书，她也可以卖掉。

好吧，一个月已经过了快三分之一，我还没开动。之前都去干嘛了呀？

以下是我的回忆流水帐：

我先是沉迷于 Github 上 Ada 学院的编程课程，发现太多东西不懂，为了打基础，去 Codecademy 上把 Command Line 的课程过了一遍。接下来的课程是学习使用 Git，可不知道为什么，我突然对乔布斯产生兴趣，便暂停课程，去把《乔布斯传》啃完了。

在忍受完乔布斯过山车一样的脾气后，我的兴趣又转移到了皮克斯，毕竟他们做的才是实打实的创意。读完《孵化皮克斯》，发现里面讲的都是商业运作（作者是 CFO），而不是创意。顺藤摸瓜，找到皮克斯的创始人之一艾德·卡特姆写的《创新公司：皮克斯的启示》，才算找对书，看了头两章，确实对创意人员的管理有很多借鉴之处。然而，这本书让我莫名进入了一种「工作状态」，我可是无业游民哦，还是等工作以后再看吧，暂缓之。

之后，我在网上冲浪时，发现一个名叫无产阶级图书馆的电子资源网站，兴致勃勃下了小几十本书，忍不住开始看（完全忘记卢奶奶给我的任务了）。先是看了一本政治八卦书，写得像校园小说，有些地方明显在胡说八道，浪费我的好奇心；随后是《暗网》，看了几章，没多大意思，弃；最后开读张纯如的《美国华人史》，这是本好书！

读完《美国华人史》，忽然想起卢奶奶布置的任务，赶紧把本月要看的书搬了出来，整理如下：

  1. 《读库 1901》
  2. 《读库 1902》
  3. 《读库 1903》
  4. 《读库 1904》
  5. 《读库 1905》
  6. 《读库 1906》
  7. 《一课经济学》
  8. 《新黑暗时代》

读库 19 年系列，其实是去年的书债了，也不知怎的，书里的一些文章，当时就是看不下去，所以今年也没有续订。虽然有 6 本，但部分文章去年已经看过了，只要补补漏，把之前落下的篇章看完，估计实际的阅读量在 3 本书左右。

《一课经济学》，想看了好几年，这段时间却没啥欲望看它。不行，作为奥地利经济学派的入门书，还是得看。

最后一本《新黑暗时代》，是某个周末跟卢奶奶出门散步，特地去图书馆借的，总得在到期前看完吧，不枉那次走一遭。

好了，梳理完毕，去看书咯。
